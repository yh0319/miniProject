Zeppelin 란?

하둡이나 스파크나 결과적으로 CLI(Command Line Interface, 사용자 기반 명령어 인터페이스) 기반이기 때문에 실제 데이터 분석 업무 시 여러가지 알고리즘이나
변수값을 바꿔가며 분석 결과를 반복시켜 그래프나 테이블같은 시각화를 제공해야 하는데, CLI 기반의 도구로는 불편하다.
아파치 제플린은 스파크를 통한 데이터 분석의 불편함을 웹 기반의 노트북을 통해 해결하고자 만들어진 어플리케이션이다.
제플린을 이용해 Python같은 언어를 사용해 분석 코드를 짤 수 있고 이 결과를 시각화하여 볼 수 있는 큰 장점이 있다.
현재 제플린은 스파크뿐 아니라 Livy, SQL 등 다른 데이터 분석 도구나 데이터베이스에 접근하여 쿼리하는 것을 쉽게 할 수 있는 확장 기능들을 지원한다.
제플린 하나만으로도 다양한 시스템의 각 요소들에 자유롭게 접근하여 데이터를 다룰 수 있다는 확장성 떄문에 비교적 최근에 나온 분석도구임에도 많이 사용되어지고 있다.
